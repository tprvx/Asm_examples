# asm_examples
```
rax - accumulator
rbx - base (pointer of data)
rcx - counter
rdx - data (division use)
rip - instruction's pointer
rsp - stack pointer

mov => move
cmp => compare
je  => jump equal

rax - 64bit
eax - 32bit
ax  - 16bit
ah/al - 8bit
```
### build
```
sudo apt install fasm
cd /hello_world
../make once run
```
